📍 Spotlight Focus Tool

Drag this blue button below to your bookmark bar: then click it to activate, controls below

Spotlight

Controls

  • Mouse → move the spotlight
  • C → Circle mode
  • L → Line mode
  • ← / → → Bigger/Smaller
  • Shift + ←/→ → Darker/Lighter
  • D → Toggle Dark Mode
  • S → Toggle Spotlight On/Off
  • Esc → Exit
